## 2. 디자인 가이드 (v2)

### 2.1 톤 & 무드
- **캐주얼 미니멀** — 10~30대 SNS 세대 타겟
- 원본 스크린샷 톤 그대로: 모노크롬(B&W) + 클린 + 여백 많이
- 장식 요소 최소화, 콘텐츠 중심
- 마이크로카피는 친근하고 가볍게 ("양말 하나가 누군가를 따뜻하게 해요")

### 2.2 컬러
```
Primary:     #1A1A1A (거의 블랙)
Background:  #FFFFFF (화이트)
Surface:     #F5F5F5 (라이트 그레이)
Border:      #E0E0E0 (가벼운 그레이 — v1의 두꺼운 블랙 보더 X)
Text:        #1A1A1A / #666666 (서브)
Accent:      없음 (모노크롬 유지)
```

### 2.3 타이포그래피
- 한글: Pretendard (system fallback)
- 영문 상품명: **소문자** 사용 (black socks, white socks)
- 웨이트: Regular(400) + Bold(700) 2종만
- 가격: Bold, 숫자 강조

### 2.4 컴포넌트 스타일
- **버튼**: 라운드 코너(12~16px), 블랙 fill(Primary) / 아웃라인(Secondary)
- **카드**: 얇은 보더(1px #E0E0E0), 라운드 코너(8px)
- **여백**: 넉넉하게 — padding 24px+, gap 16px+
- **상품 카드**: 두꺼운 블랙 보더(2px), 소문자 영문명, 중앙 정렬

### 2.5 레이아웃
- 모바일 퍼스트 (단일 컬럼)
- PC: max-width 1200px, 중앙 정렬
- 상품 영역 + 장바구니/결제를 PC에서 2컬럼 배치

## 3. 상품 구조

### 3.1 상품 목록 (2종만)

| 상품 | 색상 | 사이즈 | 가격 | 재고 | 기부 |
|------|------|--------|------|------|------|
| white socks | 화이트 | 원사이즈 | 2,990원 | 750켤레 | 1:1 (750켤레) |
| black socks | 블랙 | 원사이즈 | 2,990원 | 750켤레 | 1:1 (750켤레) |

### 3.2 구매 흐름
1. 메인 → 상품 선택 (흰/검)
2. 수량 선택 (1~재고 범위)
3. 장바구니 → 결제
4. 결제 완료 → 기부 인증서 + 기부통 업데이트

### 3.3 재고 표시
- 각 상품에 "남은 수량: 723/750" 형태로 표시
- 재고 적으면 "🔥 마감 임박!" 뱃지
- 0되면 "SOLD OUT" 처리

## 4. 흑/백 점유율 비교 UI (신규)

### 4.1 컨셉
- 흰색 vs 검정색 양말의 판매 경쟁 시각화
- "지금 더 인기있는 색은?" → 게이미피케이션 요소
- SNS 공유 유도 ("나는 #블랙팀 🖤" / "나는 #화이트팀 🤍")

### 4.2 UI 구성

#### 메인 페이지 — 실시간 비율 바
```
[========== 흰색 52% ==========|======== 검정 48% ========]
         382켤레 판매                   368켤레 판매
```
- 가로 프로그레스 바, 흑/백 양쪽에서 채워지는 형태
- 실시간 퍼센트 + 판매 수량 표시
- 구매할 때마다 바가 움직이는 애니메이션

#### 기부 현황 페이지 — 상세 비교
- **일별/주별 추세 차트** (간단한 라인 차트 or 바 차트)
- 누적 판매량 비교
- "오늘의 승자" 표시
- 최근 구매 피드 (실시간 느낌)

### 4.3 데이터
- 데모이므로 실제 DB 대신 zustand 스토어 + 더미 데이터
- 초기값: 흰색 350, 검정 320 (이미 판매된 것처럼)
- 유저 구매 시 실시간 반영

## 5. 페이지별 상세

### 5.1 메인 (랜딩)
- 히어로: "한 켤레를 사면, 한 켤레가 기부됩니다" (유지)
- **흑/백 점유율 비율 바** (신규)
- 상품 2종 카드 (흰색/검정)
- 기부 카운터 (sticky 상단 바)
- 브랜드 스토리 티저

### 5.2 상품 상세
- 상품 이미지 (🧦 이모지 → 색상별 구분)
- 가격 + 남은 수량 표시
- 수량 선택 → "N켤레 기부됩니다" 실시간 표시
- 장바구니 담기 버튼

### 5.3 장바구니
- 아이템 리스트 + 수량 조절
- 주문 요약 (금액 + 기부량)
- 결제하기 버튼

### 5.4 결제 (체크아웃)
- 배송 정보 입력
- 결제 방법 (데모)
- 주문 요약

### 5.5 주문 완료
- 기부 인증서 카드
- SNS 공유 버튼
- "나는 #블랙팀/#화이트팀" 공유 메시지

### 5.6 기부 현황
- 전체 기부 카운터 (애니메이션)
- **흑/백 상세 비교 UI** (신규)
- 기부처 정보

### 5.7 브랜드 스토리
- 1구매=1기부 철학
- 양마르 소개

## 6. 기술 스택

- Next.js 15 (static export, GitHub Pages)
- Tailwind CSS
- zustand (상태 관리)
- Framer Motion (애니메이션)
- **no** Three.js, **no** next/link (GitHub Pages 호환)
- StaticLink 컴포넌트 (basePath 자동 적용)

## 7. 제약사항 (GitHub Pages)

- `output: 'export'` + `basePath: '/yangmar-demo'`
- `trailingSlash: true`
- `useRouter()` 사용 금지 → `window.location.href` 사용
- `next/link` 사용 금지 → `StaticLink` (`<a>` 태그) 사용
- `.nojekyll` 필수
- 빌드: `./node_modules/.bin/next build` (npx 사용 금지, 버전 충돌)
