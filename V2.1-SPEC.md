## 1. v2.1 추가 기능 (3종)

### 1.1 🏆 흑/백 기부 공약 (Pledge System)

**컨셉:** 흑팀/백팀 경쟁에 "공약"을 걸어서 구매 동기를 만든다.

**UI:**
- 메인 페이지 RatioBar 하단에 공약 카드 2개 나란히 배치
- 각 카드:
  ```
  ┌─────────────────────────┐  ┌─────────────────────────┐
  │ 🖤 블랙팀 공약            │  │ 🤍 화이트팀 공약          │
  │                         │  │                         │
  │ "블랙이 이기면            │  │ "화이트가 이기면           │
  │  추가 100켤레를           │  │  지역 양로원에             │
  │  아동센터에 기부합니다"    │  │  직접 방문 전달합니다"     │
  │                         │  │                         │
  │ 현재 52% 🔥              │  │ 현재 48%                 │
  │ [ 블랙 양말 사러가기 → ]  │  │ [ 화이트 양말 사러가기 → ] │
  └─────────────────────────┘  └─────────────────────────┘
  ```
- 마감 카운트다운 타이머 ("D-14 남음")
- 이기고 있는 팀에 🔥 뱃지

**데이터:**
```ts
interface Pledge {
  team: 'white' | 'black'
  title: string       // "추가 100켤레 아동센터 기부"
  description: string // 상세 설명
  deadline: string    // "2026-04-10"
}
```
- zustand 스토어에 pledges 배열 + deadline 저장
- 데모이므로 하드코딩 데이터

### 1.2 💬 한 줄 응원 메시지 (Cheer Wall)

**컨셉:** 구매할 때 짧은 응원 메시지를 남긴다. 기부현황 페이지에 실시간 메시지 월로 노출.

**UI — 결제 완료 페이지:**
- 기부 인증서 하단에 메시지 입력 영역
  ```
  ┌─────────────────────────────┐
  │ 💬 한 마디 남기기 (선택)      │
  │                             │
  │ [따뜻한 겨울 보내세요!    ] │  ← 20자 제한, placeholder
  │                             │
  │ 🧦  🤍  ❤️  🔥  ✨  👍      │  ← 이모지 프리셋 선택 (1개)
  │                             │
  │ [ 메시지 남기기 ]            │
  └─────────────────────────────┘
  ```
- 프리셋 메시지 버튼들 (탭하면 자동 입력):
  - "따뜻한 겨울 보내세요!"
  - "양말 한 켤레의 기적 🧦"
  - "함께라서 가능한 나눔"
  - "작은 선물, 큰 따뜻함"

**UI — 기부현황 페이지 (Cheer Wall 섹션):**
  ```
  ┌──────────────────────────────────────┐
  │ 💬 응원 메시지                         │
  │                                      │
  │ 🧦 "따뜻한 겨울 보내세요!" — 익명      │
  │ ❤️ "양말 한 켤레의 기적" — 익명         │
  │ 🔥 "블랙팀 파이팅!" — 익명             │
  │ ✨ "함께라서 가능한 나눔" — 익명        │
  │ 🤍 "작은 선물, 큰 따뜻함" — 익명       │
  │                                      │
  │ (자동 스크롤, 최근 20개)               │
  └──────────────────────────────────────┘
  ```
- 카드 형태, 자동 스크롤 (marquee 느낌)
- 최근 순 정렬

**데이터:**
```ts
interface CheerMessage {
  id: string
  text: string       // 20자 제한
  emoji: string      // 선택한 이모지
  team: 'white' | 'black'
  createdAt: number  // timestamp
}
```
- zustand 스토어에 messages 배열
- 초기 더미 데이터 10개 정도 세팅

### 1.3 🎯 목표 달성 마일스톤 (Milestone Rewards)

**컨셉:** 흑/백 각각의 판매량이 특정 마일스톤 도달 시 전체 구매자에게 보상.

**UI — 메인 + 기부현황:**
  ```
  ┌──────────────────────────────────────────┐
  │ 🎯 마일스톤                               │
  │                                          │
  │ ✅ 100켤레  무료배송 이벤트 — 달성!         │
  │ ✅ 300켤레  한정판 스티커 동봉 — 달성!      │
  │ 🔓 500켤레  양마르 에코백 증정              │
  │    [████████████░░░░] 68%  340/500       │
  │ 🔒 750켤레  시크릿 이벤트 ??               │
  │                                          │
  └──────────────────────────────────────────┘
  ```
- 달성된 마일스톤: ✅ + 취소선 or 초록색
- 진행중 마일스톤: 프로그레스 바 + 퍼센트
- 미달성: 🔒 + 흐리게

**데이터:**
```ts
interface Milestone {
  target: number       // 목표 수량
  reward: string       // "무료배송 이벤트"
  description: string  // 상세
  achieved: boolean    // 달성 여부 (판매량 기준 자동 계산)
  secret?: boolean     // 마지막은 시크릿
}
```
- 흑/백 통합 마일스톤 (전체 판매량 기준)
- 4단계: 100 / 300 / 500 / 750 (=완판)
