## 2. 신규 기능 (v2.1 기능 포함 + 2개 추가)

### 2.1 🔗 양말 릴레이 (Sock Relay) — 신규

- **컨셉:** 구매가 다음 구매자에게 혜택을 주는 '릴레이' 형태. "따뜻함을 이어주세요"
- **UI (주문 완료 페이지):**
  ```
  ┌──────────────────────────────────┐
  │ 🔗 릴레이 성공!                       │
  │                                    │
  │ "당신의 구매로 다음 사람이 10% 할인을  │
  │  받게 됩니다. 릴레이를 이어주셔서      │
  │  고맙습니다!"                        │
  │                                    │
  │    현재 [ 128번째 ] 릴레이 진행 중!    │
  │                                    │
  │ [ #양마르릴레이 공유하기 ]             │
  └──────────────────────────────────┘
  ```
- **UI (메인 페이지):**
  - "현재 128번째 릴레이 진행 중 🔗 10% 할인 받기" 배너

### 2.2 🎰 양말 룰렛 (Sock Roulette) — 신규

- **컨셉:** 구매 후 룰렛 미니게임을 통해 내 양말이 기부될 곳을 직접 확인.
- **UI (주문 완료 페이지, 메시지 입력 후):**
  ```
  ┌─────────────────────────────┐
  │ 🎰 당신의 양말은 어디로 갈까요?   │
  │                               │
  │      [ 룰렛 돌아가는 UI ]       │
  │      (기부처 로고들)          │
  │                               │
  │         [ STOP ]              │
  └─────────────────────────────┘
  ```
- **결과 화면:**
  - "축하합니다! 당신의 양말 3켤레는 [OO 보육원]으로 갑니다!"
  - 기부처 스토리 짧게 노출 (사진 + 한 줄 소개)

## 3. 키치 & 캐주얼 리디자인 가이드

### 3.1 컬러
- **기본:** 블랙 & 화이트 유지
- **액센트 컬러:**
  - **Neon Green:** `#39FF14` (CTA, 하이라이트)
  - **Hot Pink:** `#FF69B4` (뱃지, 이벤트)
  - **Electric Blue:** `#00FFFF` (링크, 아이콘)
- **배경:** 미세한 노이즈 텍스처 또는 레트로 패턴 (선택적 사용)

### 3.2 타이포그래피
- **헤드라인:** D-DIN, Galmuri9 등 **굵고 각진 폰트** 또는 **픽셀 폰트** 사용
- **본문:** Pretendard 유지
- **스타일:** 텍스트에 얇은 `text-stroke` 또는 `drop-shadow` 효과 적용

### 3.3 컴포넌트 & 그래픽
- **버튼:** 더 각지고, `box-shadow`로 입체감. 호버 시 네온 `glow` 효과.
- **카드:** `border-image`로 그라데이션 또는 점선 보더.
- **아이콘:** Lucide 아이콘 대신 **픽셀 아이콘** (8-bit 스타일) 사용
- **그래픽:** 도트(dot), 체크(check), 지지직거리는 노이즈 패턴 배경으로 활용
- **애니메이션:** Framer Motion으로 통통 튀는(bouncy) 효과, `glitch` 효과, `marquee` 텍스트 적극 사용
